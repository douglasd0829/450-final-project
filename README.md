## 450-final-project

## Overview
This project demonstrates how machine learning models can be attacked using adversarial examples and defended using adversarial training.

## Dataset
The MNIST handwritten digit dataset was used.

## Model
A neural network classifier was built using TensorFlow and Keras.

## Attack Method
The Fast Gradient Sign Method (FGSM) was implemented to create adversarial examples.

## Defense Method
Adversarial training was used to improve model robustness.

## Technologies Used
- Python
- TensorFlow
- NumPy
- Matplotlib
- Google Colab
